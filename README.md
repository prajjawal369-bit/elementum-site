# Elementum - Responsive Landing Page Clone

A modern, pixel-perfect, and fully responsive landing page developed using **React.js** and **Vanilla CSS**. This project was built as an assignment to replicate a Figma design, focusing on modular component structures, precise layouts, and smooth user interaction.

##  Features Implemented
- **Component-Driven Architecture:** Clean separation of concerns with reusable React components (`ElementumPage.jsx`, `App.jsx`).
- **Figma Design Fidelity:** Styled from scratch matching light backgrounds, specific typography weights, custom badges (`badge-green`, `badge-pink`), and overlay elements.
- **Custom SVG Background Paths:** Replicated the artistic curved connecting lines and background waves using custom inline/data SVG formatting.
- **Floating Avatar Layout:** Positions circular team and customer profiles symmetrically across curved grid modules using advanced absolute CSS mapping.
- **Fully Responsive Design:** Dynamic `@media` breakpoint configurations tailored for smooth alignment on Desktops, Tablets, and Mobile screens.

## 🛠️ Tech Stack Used
- **Frontend Framework:** React.js (Vite)
- **Styling:** CSS3 (Flexbox, CSS Grid, Absolute Positioning)
- **Icons & Shapes:** Inline SVGs
